# csc337_final
To run this project, follow the instructions below:

1) Go to https://nodejs.org/en/download and make sure you have node.js installed

2) Make sure you have express installed (you can run the following command in the project directory)
npm install express

3) Install MongoDB https://www.mongodb.com/try/download/community

4) Install Mongosh https://www.mongodb.com/try/download/shell

5) in the project directory, run this command:
node server.js

This project runs at http://localhost:8080/
